Projects

LOST IN THE OPEN (ON GOING)

Project Overview

A 2.5D strategy-based roguelike game that combines procedurally generated biomes, character and maps with challenging and engaging turn-based combat.

My Role

My main responsibility in this project is the procedural generation of the world map. Although i have also been tasked with improving and refactoring existing code.

Key Points

Used a "blob spread" algorithm to procedurally place biomes.
Developing custom editors that aid the creation of new biomes.
Generation is perlin-noise based in a hex tile map.

Portal System

Project Overview

A portal system that has completely seamless transitions. Along with working with the physics system, the system is also optimized, which allows for multiple portals at once.

My Role

Being a solo project, i was responsible for the implementation of all code. I also developed multiple assets that helped make the scene less boring (in blender).

Key Points

Developed the system with the Universal Render Pipeline.
Portals slice all teleportable objects, which gives the illusion of objects passing through the portal.

DICE 'N BULLETS

Project Overview

A doom-like FPS that was developed for a 48h game jam. In order to reach the top of the tower, the player must roll a dice that will award random weapons or effects.

My Role

Being the team's programmer, i was tasked with not only programming all systems but also developing the required VFX's, post-processing effects and implementing animations.

Key Points

Use of Visual Effects Graph & Shader Graph.
Procedurally generated recoil.
Use of reflection propes and cube to make skyboxes.

BATTERY SEQUENCE

Project Overview

A puzzle game that challenges the player by having limited inputs. This game was made for a 7 day game jam where the theme was "Single Time Use".

My Role

I worked alongside a 3D Generalist, as such, I was responsible for the implementation of all features. Along with programming, I also developed shaders & VFX's.

Key Points

Developed with the base XR Interaction Toolkit.
Played a big role in the level and game design.
Developed VFX's and shaders.
